### 📱 О проекте

Учебный класс — это мобильное приложение, созданное для организации взаимодействия между преподавателями и учениками. Платформа позволяет преподавателям создавать виртуальные классы, наполнять их тестами и учебными материалами, а ученикам — присоединяться к классам, изучать материалы и проходить тестирование с автоматической проверкой результатов.

---

### Основные возможности

- Регистрация и авторизация с разграничением ролей (преподаватель / ученик)
- Создание учебных классов с автоматической генерацией уникального кода
- Присоединение к классу по коду (для учеников и соавторов)
- Конструктор тестов с вопросами двух типов (один ответ / несколько ответов)
- Добавление учебных материалов (текстовый конспект или ссылка)
- Редактирование и удаление собственных тестов и материалов
- Прохождение тестирования с автоматическим подсчётом результатов
- Просмотр результатов учеников преподавателем
- История попыток ученика с оценками
- Разграничение прав: создатель класса, соавтор, ученик
- Защита паролей (Firebase Authentication)

### Установка и запуск

**Требования**
- Android Studio (последняя версия)
- JDK 11 или выше
- Устройство или эмулятор с Android 8.0+
- Аккаунт Firebase

**Шаги для запуска**

1. Клонировать репозиторий
```
git clone https://github.com/DariaSvoboda/praktika2.git
cd praktika2
```

2. Настроить Firebase
- Создать проект в Firebase Console
- Добавить Android-приложение в проект (имя пакета: com.example.praktika2)
- Скачать файл google-services.json и поместить в папку app/
- Включить Authentication (Email/Password)
- Создать Firestore Database (тестовый режим, регион eur3)

3. Открыть проект в Android Studio
- File → Open → выбрать папку проекта
- Дождаться синхронизации Gradle

4. Запустить приложение
- Выбрать устройство или эмулятор
- Нажать Run

### Роли пользователей

**Преподаватель**
- Создание учебных классов
- Присоединение к классам как соавтор
- Создание тестов через конструктор
- Добавление учебных материалов
- Редактирование и удаление своих тестов и материалов
- Просмотр результатов учеников

**Создатель класса**
- Полный доступ ко всем тестам и материалам класса

**Соавтор класса**
- Создание, редактирование и удаление только своих тестов и материалов
- Чужие тесты и материалы доступны только для просмотра

**Ученик**
- Регистрация с обязательным кодом класса
- Просмотр тестов и материалов
- Прохождение тестирования
- Получение результата сразу после завершения
- Просмотр истории своих попыток
